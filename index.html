Number Game

How to play this game:

Set up your four digit secret number

    The 4 digit number may not start with a 0 and it can not have repeating digits.

Follow the <==

1. Press the "start" button

2. Guess the computer's secret number

    Input a 4 digit number and press enter.

    The computer will return your clues in your clue list on the bottom.

3. Computer guesses your secret number

    If any of the numbers that the computer guesses matches any of your secret numbers but aren't in the same spot, return a B.

    Otherwise, return an A if they are in the same spot.

    Enter how many A's and B's there are and then press the next round button.

4. Repeat steps 2 and 3 until you, or the computer, win by guessing the other's secret number!
